Step 1 — Create a GitHub PAT

Go to github.com/settings/tokens/new
Select Classic token → tick gist scope only → click Generate
Copy the token (starts with ghp_)

Step 2 — Paste it here

Your token is stored only in this browser's localStorage. It is sent exclusively to api.github.com to read/write a private Gist that holds your data. No third-party servers. Ever.
